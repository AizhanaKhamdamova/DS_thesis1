# Нейродифференциальные уравнения в задачах управления динамическими системами

## О проекте

Данный репозиторий содержит исследовательский прототип системы, разработанный в рамках НИР по теме:

**«Нейродифференциальные уравнения в задачах управления динамическими системами»**

Цель проекта — исследовать возможность восстановления непрерывной динамики нелинейной системы по наблюдаемым траекториям с помощью **Neural Ordinary Differential Equations (Neural ODE)**.

В текущей версии реализован работающий прототип, который:
- задаёт эталонную трёхмерную динамическую систему;
- генерирует обучающие данные численным решением ОДУ;
- обучает нейросетевую аппроксимацию правой части системы;
- сохраняет лучшую модель;
- визуализирует траектории и кривую обучения;
- тестирует модель на новом начальном условии.

Проект соответствует требованиям задания **«Прототип системы с документацией»**: содержит код, инструкцию по запуску, результаты первичного тестирования, описание реализованного функционала и элементы инженерного оформления проекта.

## Постановка задачи

Рассматривается нелинейная динамическая система в трёхмерном пространстве:

\[
\frac{dx}{dt} = -\alpha(r^2 - R^2)x - \omega y + \kappa z
\]

\[
\frac{dy}{dt} = -\alpha(r^2 - R^2)y + \omega x
\]

\[
\frac{dz}{dt} = -\beta z + \eta x
\]

где

\[
r^2 = x^2 + y^2
\]

Эта система используется как эталонная. По сгенерированным траекториям обучается модель **Neural ODE**, которая должна приблизить неизвестную правую часть динамической системы.

## Реализованный функционал

В прототипе реализованы следующие компоненты:

1. **Генерация эталонной траектории**
   - задана истинная 3D-система;
   - траектория вычисляется численным методом `dopri5`.

2. **Подготовка данных для обучения**
   - из полной траектории формируются короткие временные окна;
   - используется батчевое обучение на фрагментах траектории.

3. **Нейросетевая модель**
   - реализована модель `ODEFunc3D` на базе полносвязной нейронной сети;
   - сеть получает текущее состояние \((x, y, z)\) и возвращает оценку \((dx/dt, dy/dt, dz/dt)\).

4. **Обучение Neural ODE**
   - используется библиотека `torchdiffeq`;
   - применяется `odeint_adjoint`;
   - оптимизация выполняется алгоритмом Adam.

5. **Сохранение лучшей модели**
   - лучшая модель сохраняется в файл `best_model_3d.pth`.

6. **Визуализация**
   - графики координат во времени;
   - 3D-траектория;
   - фазовая проекция на плоскость \((x, y)\);
   - срез изученного векторного поля;
   - кривая обучения.

7. **Первичное тестирование**
   - модель проверяется на новом начальном условии;
   - рассчитывается тестовая ошибка.
